# Wall Street Agent

AI-leveraged retail trading system. Built transparently, in public.

## The Thesis

We're not competing with Citadel. We're a retail trader with frontier AI, operating in markets institutions won't touch (compliance blocks them), collecting crumbs that are life-changing at our scale. Every iteration of the frontier model closes the gap between a $2M/year quant team and one person with Claude.

- **Python** MCP bridge for agent integration
- **Claude/GPT** as thinker tier for trade evaluation
- **128GB+ VRAM** local hardware for 24/7 operation

## Consortium Pressure Test

Five frontier models (Claude, GPT-5.4, Gemini 2.5 Pro, Grok 4, DeepSeek v3.1) unanimously pressure-tested the strategy in a two-round roundtable. They called the return projections delusional, the scope too broad, and half the modules mirages. We incorporated their risk framework (unit economics, detected vs captured edge, mandatory risk infrastructure) and rejected their institutional-lens criticism. Full transcript in `data/consortium/`.
